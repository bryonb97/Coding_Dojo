@media queries

CSS uses the @media rule to detect information about the device being used to access the website.For example, it can be used to detect the size of the device, its capabilites, and its orientation. For this course, we will be using it to the detect the device size, but keep in mind that it is capable of muchmore.

Desktop: Width greater than or equal to 1024px
